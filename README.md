# 苍梵界跑团房规

`cfj-sanity-system` 是苍梵界 DND5e 跑团使用的房规模块。当前版本主要处理理智规则：SAN 检定、SAN 豁免、理智损失、理智状态、裂解/崩溃症状、短休和长休恢复。

模块不再要求把多个宏按钮摆在外面。GM 的聊天输入框上方会出现一整行 **跑团房规** 按钮；玩家端不会显示这个按钮。GM 点开后的控制台、DC、同源、熟练、主动深入和目标选择都只对 GM 可见。DM 发起后，在线目标玩家会收到强制判定弹窗和自己的判定卡；离线玩家不会生成本次 pending 请求。

## 安装地址

在 Foundry 的模块安装器中填写：

```text
https://raw.githubusercontent.com/knightinrain/cfj-sanity-system/main/module.json
```

## 安装步骤

1. 打开 Foundry VTT 管理后台。
2. 进入 **Add-on Modules / 附加模块**。
3. 点击 **Install Module / 安装模块**。
4. 粘贴上面的 Manifest URL。
5. 安装模块。
6. 进入世界，在 **Manage Modules / 管理模块** 中启用 **苍梵界跑团房规**。
7. 刷新一次世界页面。

## 日常使用

### DM

1. 在聊天输入框上方点击 **跑团房规**。
2. 聊天区会出现只对 GM 可见的 **苍梵界跑团房规控制台**。
3. 点击 **发起理智判定**。
4. 设置类型、DC、同源、是否加入熟练、是否主动深入，并勾选目标角色。
5. 发送后，每个目标角色必须有在线的非 GM 拥有者。在线目标玩家会收到强制判定弹窗和自己的判定卡；GM 也能看到这些卡。没有在线玩家拥有者的角色会被跳过。

`/理智`、`/san`、`/sanity` 仍保留为 GM 备用入口，但不再是主要操作方式。

### 玩家

1. 等 DM 发起理智判定。
2. 收到强制判定弹窗后，点击 **进行理智豁免** 或 **进行理智检定**。第一个弹窗必须完成掷骰才能关闭。
3. 不需要设置 DC，也看不到 DC 设置项。掷骰完成后，会收到一次性结果弹窗，可以手动关闭。
4. 也可以点角色卡上的 SAN 检定 / SAN 豁免，效果相同。

### 建卡或旧卡同步

1. GM 选中角色 token。
2. 点击聊天区上方的 **跑团房规**。
3. 点击 **初始化或刷新选中角色**。
4. 选择：
   - **生成新理智值**：按 `4d6kh3` 生成最大理智和当前理智。
   - **只连接/刷新**：不重掷，只同步当前理智、最大理智、状态和资源栏。

## 推荐设置

- `默认 DC`：15
- `玩家必须等待 GM 发起`：开启
- `理智显示资源栏`：主资源栏；如果角色已有用途，改用副资源栏、第三资源栏或不写入资源栏
- `短休自动处理理智`：开启
- `长休自动处理理智`：开启
- `自动生成裂解/崩溃症状`：开启
- `显示 GM 明细`：开启

## 理智损失

理智豁免失败后才降低当前理智。

| 结果 | 当前理智变化 |
| --- | --- |
| 成功 | 不降低 |
| 失败 | -1 |
| 失败 5 点或更多 | -2 |
| 失败 10 点或更多，或自然 1 | -3 |
| 主动深入 | 在上述基础上额外 -1 |

自然 20 成功。自然 1 按最强失败损失处理。

## 理智状态

| 已损失理智 | 状态 | 效果 |
| --- | --- | --- |
| 0-1 | 稳定 | 无影响 |
| 2-4 | 动摇 | 下一次同源相关检定或豁免的结果减去 1d4 |
| 5-7 | 失衡 | 对同源理智豁免具有劣势；不能从同源现象获得优势 |
| 8-10 | 裂解 | 进入该状态时获得 1 个裂解症状 |
| 11+ 或当前理智为 0 | 崩溃 | 进入该状态时获得 1 个崩溃症状 |

动摇和失衡只使用状态效果。只有裂解和崩溃会生成症状。

## 休息规则

- 短休：移除裂解症状，然后重新计算理智状态。
- 长休：恢复 1 点当前理智，移除理智症状，然后重新计算理智状态。
- 当前理智不会超过最大理智。
- 休息后的重新计算不会因为角色仍在裂解或崩溃而立刻再生成一个新症状。

## 验证清单

启用模块后，用测试角色验证，验证后删除测试角色：

- GM 端聊天输入框上方出现一整行 **跑团房规** 按钮。
- 玩家端不显示这个按钮。
- GM 点击按钮后，控制台聊天卡只对 GM 可见。
- 场景工具栏里不再出现单独的理智按钮。
- GM 发起一个非默认 DC 的理智判定。
- 目标玩家收到自己的判定卡，并能点击卡内按钮掷骰。
- 非目标玩家看不到该判定卡。
- 玩家掷骰时不能编辑 DC。
- 失败后当前理智降低，角色卡 SAN 数值变化。
- 进入裂解或崩溃时，角色获得可见状态和症状效应。
- 短休移除裂解症状。
- 长休恢复 1 点当前理智并移除理智症状。
